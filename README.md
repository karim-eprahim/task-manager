# Task Manager

A modern Task Management application built with Nuxt 4.

## Tech Stack

- Nuxt 4
- TypeScript
- Pinia
- Nuxt UI
- Tailwind CSS

## Features

- View tasks
- Add task
- Edit task
- Delete task
- Search by title
- Filter by status
- Due date validation
- Loading state
- Error state
- Empty state
- Dark / Light mode
- English / Arabic support
- Responsive design

## Installation

```bash
npm install
npm run dev
